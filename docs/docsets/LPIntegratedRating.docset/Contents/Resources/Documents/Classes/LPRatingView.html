LPRatingView

This view contains a title: UILabel, an approvalButton: UIButton, and rejectionButton: UIButton.

This is the main view which will display all the information according to the configuration set, using LPRatingViewConfiguration. This can be placed anywhere a UIView might be placed. However, it is highly recommended to integrate it into a table view or collectionview. Pop ups annoy the user and lead to lower app reviews. For easy integration check out LPRatingTableViewCell, and LPRatingCollectionViewCell which make it easy to use this view with either a UITableView or UICollectionView.
